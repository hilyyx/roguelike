# roguelike

### Описание

Наш проект — это roguelike-игра, в которой игрок исследует подземелья, сражается с противниками и старается выжить как можно дольше. Для жанра характерны случайная генерация уровней, пошаговое или тактическое принятие решений и постепенное усложнение игры, из-за чего каждое прохождение отличается от предыдущего.
